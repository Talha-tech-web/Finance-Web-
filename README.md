# 💰 FinLedger

A modern **Finance Management System** built with **HTML, CSS, JavaScript, and Supabase**. FinLedger helps businesses manage income, expenses, and financial records through an intuitive dashboard with real-time database integration.

---

## 📖 Overview

FinLedger is a lightweight finance tracking application designed for small businesses. It allows users to manage transactions, monitor financial summaries, search records instantly, and export transaction data.

---

## ✨ Features

* 📊 Dashboard with financial summary cards
* ➕ Add new transactions
* ✏️ Edit existing transactions
* 🗑️ Delete transactions
* 🔍 Live search and filtering
* 📄 Export transactions to CSV
* 📈 Real-time summary (Income, Expenses, Balance)
* 📋 Transaction activity table
* ☁️ Supabase database integration
* 📱 Responsive design

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/FinLedger.git
cd FinLedger
```

---

### 2. Create a Supabase Project

1. Log in to Supabase.
2. Create a new project.
3. Open **Project Settings → API**.
4. Copy:

   * Project URL
   * Anon/Public API Key

---

### 3. Configure Supabase

Open:

```text
js/config.js
```

Replace:

```javascript
const SUPABASE_URL = "YOUR_SUPABASE_URL";
const SUPABASE_ANON_KEY = "YOUR_SUPABASE_ANON_KEY";

const TABLE_NAME = "finance";
```

---

### 4. Create the Database Table

Run the following SQL in the **Supabase SQL Editor**:

```sql
CREATE TABLE public.finance (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    category TEXT,
    date DATE,
    time TEXT,
    amount NUMERIC,
    method TEXT,
    type TEXT,
    customer_supplier TEXT,
    product TEXT,
    quantity INTEGER,
    invoice_no TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

---

### 5. Enable Row Level Security (RLS)

```sql
ALTER TABLE public.finance ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow Public Access"
ON public.finance
FOR ALL
USING (true)
WITH CHECK (true);
```

---

### 6. Run the Application

Open the project using **Live Server** in VS Code.

Example:

```text
http://127.0.0.1:5500/index.html
```
